# Maysaa Transposition Cipher

A complete Columnar Transposition Cipher implementation for System Security.

## Features
- Encryption/Decryption
- Brute-force & Heuristic Attacks
- AI Key Recommender
- Word/Phrase Dictionary
- Visual Web Interface

## Setup
```bash
pip install -r requirements.txt
python app.py
```

Visit http://localhost:5000
